# InnovateX – Digital Solutions Website

## 📌 Project Overview

InnovateX is a modern, responsive landing page designed for a digital solutions company.
The website showcases services, company information, and a contact form while focusing on clean UI design and responsive layout.

This project demonstrates basic **frontend web development skills using HTML, CSS, and JavaScript**.

---

## 🚀 Features

* Responsive navigation bar
* Modern hero section with gradient background
* Smooth scrolling navigation
* Interactive service cards with hover effects
* About section describing the company
* Contact form with basic validation
* Mobile-friendly responsive design
* Clean and modern UI using Google Fonts

---

## 🛠 Technologies Used

* **HTML5** – Website structure
* **CSS3** – Styling and layout
* **JavaScript** – Interactivity and form validation
* **Google Fonts (Poppins)** – Modern typography

---

## ⚙️ How to Run the Project

1. Download or clone the repository.
2. Open the project folder.
3. Double-click **index.html**
   or open it using any modern web browser.

No server or installation required.

---

## 📱 Responsive Design

The website is designed to work on:

* Desktop computers
* Tablets
* Mobile devices

CSS media queries ensure proper layout across screen sizes.

---

## 📬 Contact Form

The contact form includes basic JavaScript validation to ensure:

* Name field is filled
* Email is valid
* Message field is filled

Currently the form works in **demo mode** and does not send data to a backend server.
