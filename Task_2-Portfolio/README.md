# 🌐 K. Lakshmi Kameswari – Developer Portfolio

## 📌 Project Overview

This project is a **personal portfolio website** built to showcase the skills, projects, and services of **K. Lakshmi Kameswari**, a passionate web developer.

The website highlights professional experience, completed projects, and contact information in a clean and modern interface.

It is designed to be **responsive, interactive, and visually appealing** for recruiters and potential clients.

---

## 🚀 Features

* Responsive modern portfolio layout
* Animated **typing text effect**
* **Counter animation** for experience and projects
* Smooth scrolling navigation
* Interactive **hover effects**
* Modern **dark themed UI**
* Mobile responsive design
* Scroll reveal animations
* Scroll-to-top button

---

## 🛠 Technologies Used

* **HTML5** – Structure of the website
* **CSS3** – Styling, animations, responsive design
* **JavaScript (Vanilla JS)** – Interactivity and animations
* **Google Fonts (Poppins)** – Modern typography

---

```
portfolio-website
```

## ⚙️ How to Run the Project

1. Download or clone the repository.
2. Open the project folder.
3. Double-click **index.html** or open it in any web browser.

No server setup or installation is required.

---

## 📱 Responsive Design

The portfolio is optimized for:

* Desktop computers
* Tablets
* Mobile devices

Media queries ensure proper layout on all screen sizes.

---

## 📊 Portfolio Sections

The website includes the following sections:

* **Home** – Introduction and developer role
* **About** – Information about the developer
* **Services** – Skills and services offered
* **Projects** – Portfolio projects with live links
* **Contact** – Email and social profiles
